# IMPORTANT: Download Required Model File

To use hand tracking, you need to download the MediaPipe Hand Landmarker model:

1. Download the model from:
   https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/1/hand_landmarker.task

2. Place the downloaded `hand_landmarker.task` file in this directory:
   `app/src/main/assets/`

The file should be approximately 11-12 MB in size.
